# ООО «Мебельное ателье» — программа учета товаров

## Запуск
```bash
pip install -r requirements.txt
python app.py
```

Логин по умолчанию: `admin`  
Пароль по умолчанию: `admin123`

## Возможности
- SQLite база данных создается автоматически при первом запуске.
- Материалы загружаются из `data/materialy.csv`, подготовленного из переданного Excel-файла.
- Разделы: материалы, поставщики, поступление, расход, отчеты.
- Добавление, редактирование и удаление записей.
- Расход с зависимыми списками: шкафы, кухни, фасады.
- Отчеты можно сохранить в `.txt` и отправить на печать.
- Логотип компании отображается сверху.
- Есть переключение светлой/темной темы.

## Обновление: расчет расхода листов

Во вкладке «Расход» теперь выбирается изделие и вводится количество изделий. Программа автоматически считает списание листов по нормам:
- Кухня прямая 3 метра — 5 листов
- Угловая кухня 5 метров — 6 листов
- Шкаф купе — 3 листа
- Одежный шкаф — 1 лист
- Шкаф для документов — 1 лист
- Стол — 1 лист
- Тумба — 0.25 листа
- Комод — 1 лист

Если листов на остатке не хватает, программа показывает предупреждение и не списывает расход.
