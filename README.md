## 📌 О проекте

<b>Система учёта материалов мебельного ателье</b> — настольное приложение для автоматизации складского учёта на мебельном производстве.

Программа позволяет хранить информацию о материалах и поставщиках, регистрировать поступления, рассчитывать расход материалов при изготовлении мебели, контролировать текущие остатки и формировать отчёты.

Приложение разработано на языке <b>Python</b>. Для создания графического интерфейса используется библиотека <b>CustomTkinter</b>, а данные хранятся в локальной базе <b>SQLite</b>.

## 🎯 Цель проекта

Основная цель проекта — автоматизировать учёт материалов мебельного предприятия и заменить ручное ведение информации в таблицах и бумажных документах.

Приложение помогает решать следующие задачи:

<ul>
  <li>вести единый справочник материалов;</li>
  <li>хранить контактные данные поставщиков;</li>
  <li>регистрировать поступление материалов на склад;</li>
  <li>автоматически рассчитывать расход материалов;</li>
  <li>контролировать актуальные складские остатки;</li>
  <li>предотвращать списание материалов при недостаточном остатке;</li>
  <li>формировать отчёты по складским операциям;</li>
  <li>экспортировать отчётные данные в PDF.</li>
</ul>

## 📦 Управление материалами

Раздел «Материалы» предназначен для ведения складского справочника.

Для каждого материала можно хранить:

<ul>
  <li>наименование;</li>
  <li>производителя;</li>
  <li>цвет;</li>
  <li>код цвета или артикул;</li>
  <li>размер листа;</li>
  <li>толщину;</li>
  <li>количество на складе;</li>
  <li>дополнительное примечание.</li>
</ul>

Администратор может добавлять, изменять и удалять записи. При добавлении нового материала программа предлагает ранее использованные значения для быстрого заполнения полей.

Также реализована проверка дубликатов. Если материал с такими же основными параметрами уже существует, программа предупреждает пользователя и не создаёт повторную запись.

## 🚚 Управление поставщиками

В отдельном разделе хранится информация о поставщиках материалов.

Для каждого поставщика можно указать:

<ul>
  <li>название организации;</li>
  <li>номер телефона;</li>
  <li>электронную почту;</li>
  <li>адрес;</li>
  <li>дополнительное примечание.</li>
</ul>

Телефон автоматически приводится к формату:

```text
+7(999)123-45-67
```

Программа проверяет корректность введённого номера и предотвращает создание поставщиков с повторяющимися данными.

Из контекстного меню можно открыть создание письма поставщику через:

<ul>
  <li>стандартную почтовую программу;</li>
  <li>Gmail;</li>
  <li>Яндекс Почту;</li>
  <li>Mail.ru.</li>
</ul>

## 📥 Учёт поступлений

Раздел «Поступление» используется для регистрации новых материалов, поступивших на склад.

Пользователь последовательно выбирает:

<ol>
  <li>производителя;</li>
  <li>наименование материала;</li>
  <li>цвет;</li>
  <li>количество;</li>
  <li>стоимость;</li>
  <li>примечание.</li>
</ol>

После сохранения операции:

<ul>
  <li>создаётся запись о поступлении;</li>
  <li>фиксируются дата и время операции;</li>
  <li>количество материала автоматически прибавляется к текущему остатку;</li>
  <li>операция появляется в журнале поступлений.</li>
</ul>

Программа также проверяет повторные операции и предупреждает пользователя, если аналогичное поступление уже было зарегистрировано.

## 📤 Расчёт и списание материалов

Раздел «Расход» используется для списания материалов, которые были использованы при изготовлении мебели.

В программе предусмотрены нормы расхода для разных изделий:

<table>
  <tr>
    <th>Изделие</th>
    <th>Норма расхода</th>
  </tr>
  <tr>
    <td>Прямая кухня длиной 3 метра</td>
    <td>5 листов</td>
  </tr>
  <tr>
    <td>Угловая кухня длиной 5 метров</td>
    <td>6 листов</td>
  </tr>
  <tr>
    <td>Шкаф-купе</td>
    <td>3 листа</td>
  </tr>
  <tr>
    <td>Одежный шкаф</td>
    <td>1 лист</td>
  </tr>
  <tr>
    <td>Шкаф для документов</td>
    <td>1 лист</td>
  </tr>
  <tr>
    <td>Стол</td>
    <td>1 лист</td>
  </tr>
  <tr>
    <td>Тумба</td>
    <td>0,25 листа</td>
  </tr>
  <tr>
    <td>Комод</td>
    <td>1 лист</td>
  </tr>
</table>

Общий расход рассчитывается автоматически:

```text
Количество изделий × Норма расхода на одно изделие
```

Перед списанием программа сравнивает рассчитанный расход с текущим складским остатком.

Если материала недостаточно, операция отменяется и пользователь получает предупреждение. Это позволяет избежать появления отрицательных остатков.

## 🔍 Фильтрация таблиц

В таблицах реализованы фильтры, похожие на фильтры электронных таблиц.

Пользователь может:

<ul>
  <li>отбирать строки по значениям выбранного столбца;</li>
  <li>искать нужное значение внутри списка;</li>
  <li>выбирать сразу несколько значений;</li>
  <li>показывать только нулевые или ненулевые остатки;</li>
  <li>отбирать значения больше или меньше указанного числа;</li>
  <li>сбрасывать фильтр отдельного столбца.</li>
</ul>

Активный фильтр отмечается специальным индикатором в заголовке столбца.
